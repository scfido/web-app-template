## 使用按钮样式

使用按钮定义的样式，可以将按钮的样式应用到其他元素上。

```tsx
import { buttonVariants } from "@/components/ui/button"

<div 
    className={buttonVariants({ variant: "outline" , size: "lg" })}
>
    测试
</div>
```
